--- title: "Natural Language and Sketch Querying for Time-Series Patterns" excerpt: "A multimodal visual analytics system that combines natural language and sketch input for time-series querying." collection: portfolio permalink: /portfolio/natural-language-sketch-time-series-querying/ ---

This project explores multimodal interaction for time-series visual analytics by combining natural language queries with sketch-based pattern input. The system supports expressive pattern search and was evaluated through user studies, usage scenarios, and semantic parsing experiments.
